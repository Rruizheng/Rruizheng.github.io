# nvm安装配置

在 Windows、Mac、Linux 等操作系统中，可以使用 nvm 工具来管理 Node.js 版本，实现快速切换 Node.js 版本的目的。

# 验证是否存在

command -v nvm
1

# 安装

  1. 打开终端（Terminal），输入以下命令下载 nvm 安装脚本
curl -o- https://raw.githubusercontent.com/creationix/nvm/v0.39.0/install.sh | bash
1
  1. 如果 nvm 已经被正确安装，但是在终端中仍然无法使用 nvm 命令，则可能是因为环境变量没有被正确配置。可以按照以下步骤配置环境变量：
  • 在终端中输入以下命令打开 .zshrc 文件：
# 进入 ~/.zshrc
nano ~/.zshrc
1
2
  • 在文件末尾添加以下内容：
export NVM_DIR="$HOME/.nvm"
[ -s "$NVM_DIR/nvm.sh" ] && \. "$NVM_DIR/nvm.sh"  # This loads nvm
[ -s "$NVM_DIR/bash_completion" ] && \. "$NVM_DIR/bash_completion"  # This loads nvm bash_completion
1
2
3
  • 保存文件并退出。
  • 运行以下命令来使配置生效：
source ~/.zshrc
1

# 常用指令

  • 查看安装版本
nvm --version
1
  • 列出当前系统中已经安装的所有 Node.js 版本
nvm ls
1
  • 安装最新的稳定版 Node.js
nvm install stable
1
  • 切换到已安装的其他版本
nvm use 16.20.0
1
  • 查看可用的 Node.js 版本
nvm list available
1
  • 选择需要安装的版本
nvm install 14.17.6
1
